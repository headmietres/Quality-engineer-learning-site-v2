## 什么是过程能力指数

过程能力指数是衡量制造过程满足规格要求能力的统计指标。它将过程的实际变异与规格范围进行比较，用数值量化过程的"能力"。

## Cp 与 Cpk 的区别

**Cp（潜在过程能力指数）**：衡量过程在理想状态下（过程居中于规格中心）的能力。Cp = (USL - LSL) / 6σ，只考虑变异，不考虑偏移。

**Cpk（实际过程能力指数）**：衡量过程在实际状态下（考虑过程偏移）的能力。Cpk = min[(USL-μ)/3σ, (μ-LSL)/3σ]，同时考虑变异和偏移。

**关键区别**：如果过程完美居中，Cp = Cpk；如果过程有偏移，Cpk < Cp。两者的差值反映了改善过程中心的机会。

## 评价标准

| Cpk 值 | 过程能力评价 |
|--------|------------|
| ≥ 1.67 | 优秀（六西格玛水平） |
| 1.33 - 1.67 | 良好 |
| 1.00 - 1.33 | 一般（需监控） |
| < 1.00 | 不足（需改进） |

汽车行业通常要求 Cpk ≥ 1.33（安全/关键特性要求 ≥ 1.67）。

## 计算示例

某零件规格为 50±1mm，测量得样本均值 μ=50.3mm，标准差 σ=0.2mm。

Cp = (51-49) / (6×0.2) = 2/1.2 = 1.67

Cpk-up = (51-50.3) / (3×0.2) = 0.7/0.6 = 1.17
Cpk-low = (50.3-49) / (3×0.2) = 1.3/0.6 = 2.17
Cpk = min(1.17, 2.17) = 1.17

分析：Cp=1.67（潜在能力优秀），但 Cpk=1.17（由于过程偏移），说明需要将均值调整到 50mm 附近。

## 注意事项

1. 计算 Cp/Cpk 的前提是过程稳定且数据服从正态分布
2. 样本量至少 25-30 个子组才有统计意义
3. Cpk 反映的是短期能力，Ppk 反映的是长期能力
4. 不要只看 Cpk 数值，还要看控制图是否稳定

## 常见面试题

**Q：Cpk 和 Ppk 有什么区别？**
A：Cpk 使用组内标准差计算，反映短期能力；Ppk 使用总体标准差计算，反映长期能力。

**Q：Cp 值很高但 Cpk 值很低说明了什么？**
A：说明过程变异小（潜在能力好），但过程偏移严重，需要调整过程中心。
