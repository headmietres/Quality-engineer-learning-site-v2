# MSA 实战：测量系统分析入门

## 1.为什么要做 MSA？

​	如果你正在推行 SPC（统计过程控制），打算用数据来驱动过程改进，请先问自己一个问题：**我测出来的数据，真的可信吗？**

​	假如用一把不准的秤称重，把所有“瘦”的都标成“胖”，那后续所有的分析、调整、判断全部建立在错误基础上。**数据质量是质量管理的生命线，而 MSA（Measurement System Analysis，测量系统分析）** 就是用来评估这根生命线是否牢靠的核心工具。

> MSA 的目标：量化测量系统的变差来源，判断它是否足以支撑决策（如产品判定、过程调整）。

什么时候必须做 MSA？

- 新量具投入使用前

- 新测量方法建立时

- SPC、过程能力分析之前

- 测量系统发生变更（如更换人员、设备大修）

- 客户审核或体系认证要求（如 IATF 16949）

## 2. 测量系统的误差模型

任何测量值，都由**真实值**和**测量误差**两部分组成：
而这部分测量误差 ，又可以分解为两大维度：

- **位置误差（准确性）**：测量平均值与真值的接近程度。

  - **偏倚 (Bias)**：多次测量均值与参考值的差值。
  - **线性 (Linearity)**：在量具量程范围内，偏倚是否保持一致。
  - **稳定性 (Stability)**：随时间推移，偏倚是否发生漂移。

- **宽度误差（精密性）**：多次测量结果之间的离散程度。

  - **重复性 (Repeatability)**：同一人、同一量具、同一零件，多次测量的变异（设备自身变差，EV）。
  - **再现性 (Reproducibility)**：不同人、同一量具、同一零件，测量平均值的变异（人员变差，AV）。

  ```
          ┌────────── 偏倚 (Bias)
  位置准确度 ──├────────── 线性 (Linearity)
          └────────── 稳定性 (Stability)
  
          ┌────────── 重复性 (Repeatability)  → 设备变差 EV
  宽度精密度 ──┤
          └────────── 再现性 (Reproducibility) → 人员变差 AV
                            ↓
                合称 GR&R (Gauge R&R)
  ```

  测量系统的**总变差**来自两部分：
  总变差=工件变差+测量系统的变差
  其中 测量系统变差= 设备变差 + 人员变差，我们的理想是：测量系统变差远小于零件变差和公差，让数据能“看清”过程的变化。

## 3. 核心概念图解

以下通过一个靶心模型直观理解各术语：

- **准确且精密**：弹孔都密集在靶心 → 偏倚小，变差小。
- **准确但不精密**：弹孔围绕靶心大范围散布 → 重复性差。
- **精密但不准确**：弹孔密集但全部偏离靶心 → 偏倚大。
- **既不准确又不精密**：弹孔分散且中心偏离 → 全套测量系统不合格。

实际工作中的要求：**先精后准**。如果重复测量都乱跳，偏倚也就无从谈起。

## 4. MSA 分析的两大类型

根据测量数据的性质，MSA 分为两类，入门阶段必须重点掌握计量型 GR&R。

| 类型           | 数据特征                   | 常用方法                               | 产出                            |
| :------------- | :------------------------- | :------------------------------------- | :------------------------------ |
| **计量型 MSA** | 连续值（长度、重量、压力） | GR&R 研究（交叉/嵌套）、偏倚和线性研究 | %GR&R、ndc、偏倚、线性图        |
| **计数型 MSA** | 离散判定（OK/NG，好/坏）   | 风险分析法（Kappa 统计量、一致性比率） | Kappa 值、有效性、错判率/漏判率 |

本文后续集中讲解计量型 MSA 中最具代表性和实战性的 **GR&R 研究（量具重复性与再现性）**。

## 5. GR&R 研究实战指南（计量型）

### 5.1 前提准备

- **测量设备**已校准，分辨率（最小刻度）应至少达到公差或过程变差的 1/10。
- **选择样本零件**：从过程中抽取 10 个零件（通常要求），覆盖整个过程的实际变异范围，不要只拿连续生产的“完美件”，也不要特意选极端超差品。样本要能代表长期过程。
- **选择评价人**：通常选 2-3 名实际操作的检验员。
- **实验设计**：采用盲测和随机化。评价人不知道零件编号，测量顺序随机，避免记忆效应。

### 5.2 典型方案：3人 × 10件 × 3次（交叉设计）

设定：3 名评价人（A, B, C），10 个样本零件，每人将每个零件重复测量 3 遍。
总共获得 3×10×3 = 90 个数据点。

## 6. 判定标准与改进方向

### 6.1 通行判定基准

- **%GR&R ≤ 10%**：测量系统可接受，非常适合过程控制与分析。
- **10% < %GR&R ≤ 30%**：有条件接受，需基于重要性、成本、客户要求等因素决定是否改善。
- **%GR&R > 30%**：测量系统不可接受，必须改进。

同时要求 **ndc ≥ 5**。ndc 小于 5 意味着系统几乎无法区分零件的好坏差别，数据对过程改进毫无价值。

### 6.2 若 GR&R 不合格，如何诊断改善？

分析各分量占比：

- **如果 EV（重复性）占比大**：设备自身问题。
  - 检查量具磨损、松动、清洁度、夹持方式。
  - 更换高精度量具，或确认测量环境（振动、温度）。
  - 提升分辨率（如数显换更高位数）。
- **如果 AV（再现性）占比大**：人员间一致性差。

- - 制定标准化操作指导书（SOP），统一测量点位、力度、读数方式。
  - 加强培训并考核。
  - 使用工装夹具减少人为操作差异。
- **如果零件与评价人之间有显著交互作用**（需 ANOVA 发现）：
  - 某些人测量特定零件时系统偏倚不一致，往往指向更复杂的操作定义问题，需细化测量方法。

## 7. 其他 MSA 研究简述

### 偏倚与线性研究

- **偏倚**：对标一个已知参考值的标准件，多人多次测量，计算平均值与参考值的差，用 t 检验确认是否显著。
- **线性**：在全量程内选 5-8 个覆盖量程的标准件，评估偏倚随量程变化情况，理想情况偏倚恒定接近零。

### 计数型 MSA（属性一致性分析）

- 挑选一定数量的样品，包含一定比例的边界品（故意选取难以判定的“灰色地带”样品）。
- 让多名评价人对每个样品盲测判定两次，分析：
  - **自身一致性（重复性）**：同一个评价人两次判断是否一致。
  - **与标准一致性（有效性）**：评价人的判断是否与已知标准结果一致。
  - **Kappa 值**：去除随机一致性后的真正一致性度量。Kappa > 0.75 良好，0.4~0.75 一般，<0.4 差。

## 8. MSA 与 SPC 的黄金顺序

在质量管理工具链中，有一个颠扑不破的先后顺序：

> **先 MSA，再 SPC，后过程能力（Cpk）分析。**

- MSA 保证了数据源头的可靠。
- SPC 基于可靠数据监控过程稳定性。
- 过程能力（Cpk）只能在统计受控且测量系统合格的前提下才能计算和判断。

如果跳过 MSA 直接上 SPC，等于把噪音当作信号，把量具的波动当成过程的波动，轻则浪费调整资源，重则做出错误决策。

## 9. 常见的 MSA 错误与陷阱

1. **样本选择不当**：只挑连续好品，导致零件变异 PV 极小，GR&R% 天然偏高。**必须覆盖过程全变异**。
2. **未进行盲测**：评价人知道零件编号，凭记忆读数，严重低估再现性。
3. **测量顺序固定**：导致时间趋势（如量具预热漂移）混入再现性，人为割裂误差分量。
4. **分辨率不足**：量具最小刻度无法区分零件间微小差异，ndc 奇低。
5. **评价人不具代表性**：用工程师代替一线操作工，结果不能反映真实日常测量水平。
6. **只看 %GR&R 不看方差分量**：%GR&R 合格可能因为零件变异太大“掩盖”了测量变差，实际测量误差可能依然超过公差要求（%P/T 太大），必须两个指标同看。

## 10. 结语

MSA 不是一次性的“过关”测试，而是贯穿产品全生命周期的测量风险管理。它让我们从“我相信这个数”变成 **“我有数据证明这个数可信”**。

实战中，请从最简单的一把卡尺、三名检验员、十个零件开始，完成一次完整的 GR&R 研究。当你看到那个代表 %GR&R 的数值时，你就真正迈进了数据驱动质量改善的大门。

> 记住：没有合格的测量系统，一切统计分析和过程控制都只是“高级的猜测”。
