# GD&T 基础：几何尺寸与公差入门

## 1. 什么是 GD&T？

**GD&T** 是 **Geometric Dimensioning and Tolerancing** 的缩写，中文称为 **几何尺寸与公差**。它是一种用符号、数字和基准来表达零件几何特征要求的标准化工程语言。与传统的正负尺寸公差不同，GD&T 不仅控制尺寸，还精确控制形状、方向、位置和跳动，从而确保零件的可装配性和功能性。

**核心思想**：不是单纯说“这个孔直径 10±0.1 mm”，而是说“这个孔的位置，相对于基准 A、B、C，在最大实体状态下可以有一个直径 0.2 mm 的公差带”。

---

## 2. 为什么需要 GD&T？

传统坐标公差（± 公差）存在明显局限：

- **公差带形状单一**：只能形成矩形或正方体公差带，而 GD&T 可以使用圆形、圆柱形等更符合功能需求的公差带。
- **无法表达关联关系**：不能准确描述一个特征相对于另一个特征的位置、方向要求。
- **公差累积与理解歧义**：检具设计和功能判断容易出现争议。
- **无法利用公差补偿**：无法体现“孔做大一些，位置可以偏一点”的功能，导致不必要的废品。

**GD&T 的优势**：
- 公差带更大更合理（如圆形公差带比正方形面积大57%）
- 允许使用功能检具，提高检测效率
- 清晰传递设计意图，减少误解
- 可以定义基准顺序，模拟装配关系
- 支持最大实体原则（MMC）和最小实体原则（LMC），提供额外制造公差

---

## 3. 主要标准

目前全球有两大 GD&T 标准体系：

| 标准体系 | 代表性标准 | 特点 |
|----------|------------|------|
| **ASME**（美国） | ASME Y14.5-2018 | 强调“包容原则”，默认规则为 **Rule #1**（泰勒原则）。符号和解释与 ISO 有细微差别。 |
| **ISO**（国际） | ISO 1101:2017 等系列 | 默认独立原则，需显式标注包容要求 (E圈)。符号体系与 ASME 大部分兼容，但存在差异。 |

入门阶段通常以应用最广的 ASME 符号体系为主进行讲解，并适当指出 ISO 区别。

## 4. 基本概念

### 4.1 特征 (Feature)
零件上的特定部位，如平面、圆柱面、孔、槽等。

### 4.2 基准 (Datum)
理论上精确的点、线、平面或曲面，作为测量其他特征的参考。基准是构建坐标系的基础。

- **基准平面**：一般为第一基准（主定位面）、第二基准（次定位面）、第三基准（第三定位面）。
- **基准特征**：用于建立基准的实际零件特征。通常需要标注基准符号（字母加方框）。
- **基准目标**：在非刚性零件或铸件上，指定点、线或区域作为基准接触位置。

### 4.3 公差带 (Tolerance Zone)
允许特征变动的区域。GD&T 定义了公差带的形状、大小、方向和位置。

- **形状**：两平行平面、一个圆柱体、两同心圆等。
- **大小**：公差值，如直径 0.5 mm。
- **方向**：平行、垂直或倾斜于基准。
- **位置**：理论正确尺寸（加方框的尺寸）决定公差带中心。

### 4.4 理论正确尺寸 (Basic Dimension)
加方框的数值，如 `[50]`，表示没有直接公差的理想尺寸，用于定位公差带中心。它必须配合形位公差使用。

### 4.5 材料条件修饰符 (Material Condition Modifiers)
用于指示公差是否随特征尺寸变化而变化。

| 符号 | 名称 | 说明 |
|------|------|------|
| Ⓜ | 最大实体条件 (MMC) | 特征尺寸最大实体时（轴最大，孔最小），形位公差值按标注值。当偏离最大实体时，额外获得尺寸公差补偿。 |
| Ⓛ | 最小实体条件 (LMC) | 特征尺寸最小实体时（轴最小，孔最大），公差有效。常用于保证最小壁厚。 |
| Ⓢ | 不考虑特征尺寸 (RFS) | 形位公差值与特征实际尺寸无关，始终为标注值。ASME 默认规则（不标即 RFS），ISO 中需特殊标注。 |

### 4.6 包容原则与独立原则
- **ASME 规则 #1（包容原则）**：对于尺寸要素（如轴、孔），其形状误差受尺寸公差控制。在 MMC 时，形状必须完美，不能超出最大实体边界。
- **ISO 默认独立原则**：尺寸公差与几何公差互不相关，需单独标注包容要求（圈E）才有类似效果。

## 5. 几何公差符号一览

GD&T 将公差分为四大类：**形状**、**方向**、**位置** 和 **跳动**。下面是常用符号表（基于 ASME Y14.5）。

| 类别 | 公差类型 | 符号 | 是否需要基准 | 说明 |
|------|----------|------|--------------|------|
| 形状 | 直线度 | — | 否 | 控制直线元素（轴线或素线）的平直度 |
|      | 平面度 | ▱ | 否 | 控制表面的平整度 |
|      | 圆度 | ○ | 否 | 圆柱或圆锥任一截面的圆整度 |
|      | 圆柱度 | ⌭ | 否 | 同时控制圆柱面的圆度、直线度和锥度 |
| 方向 | 平行度 | ∥ | 是 | 相对于基准平面或轴线的平行程度 |
|      | 垂直度 | ⟂ | 是 | 相对于基准平面或轴线的垂直程度 |
|      | 倾斜度 | ∠ | 是 | 以指定理论角度（非90°）相对于基准 |
| 位置 | 位置度 | ⌖ | 是或否 | 控制特征中心、轴线或中面的准确位置（最常用） |
|      | 同心度 (同轴度) | ◎ | 是 | 控制回转体轴线与基准轴线的重合度（ASME 2018 已取消，改用位置度）|
|      | 对称度 | ⌯ | 是 | 控制特征中心面对基准中心面的对称程度（ASME 2018 已取消）|
| 跳动 | 圆跳动 | ↗ | 是 | 表面元素绕基准轴线旋转一周的最大允许跳动量 |
|      | 全跳动 | ↗↗ | 是 | 表面整体绕基准轴线连续旋转并轴向移动时的最大跳动量 |

**轮廓度** 既可以控制形状也可以控制位置，具有双重性：

| 符号 | 名称 | 说明 |
|------|------|------|
| ⌒ | 线轮廓度 | 控制任意曲线轮廓的形状（和/或位置） |
| ⌓ | 面轮廓度 | 控制任意曲面的形状（和/或位置） |

轮廓度不标注基准时控制形状，标注基准时同时控制形状和位置，是一种万能的几何公差。

## 6. 特征控制框 (Feature Control Frame)

形位公差通过特征控制框标注，结构如下：
```
┌──────┬──────┬──────┬──────┐
│ ⌖ │ Ø0.5 │ Ⓜ │ A │ B │ C │
└──────┴──────┴──────┴──────┘
```
- 第一格：几何公差符号
- 第二格：公差值，若公差带为圆形或圆柱形需加直径符号 `Ø`
- 第三格（可选）：材料条件修饰符或其它修饰符（如投影公差带 `P`）
- 后续格：基准字母（按优先顺序），每个基准后面也可带修饰符。

---

## 7. 基准的标注与三基准面体系

基准通过一个大写字母加方框、并连接实心或空心三角标识。基准建立通常遵循 **3-2-1 原则**：
- **第一基准 A**：限制3个自由度（通常为平面，接触3点）
- **第二基准 B**：限制2个自由度（直线，接触2点）
- **第三基准 C**：限制1个自由度（点，接触1点）

在图纸中基准顺序很重要，它反映了装配定位的优先级。

## 8. 常见几何公差详解及示例

### 8.1 直线度
用于控制轴的轴线弯曲或表面素线的直线度。

- **标注**：`— Ø0.08` （控制轴线）或 `— 0.1` （控制素线）
- **检测**：刀口尺、三坐标测量机（CMM）、平板与百分表。

### 8.2 平面度
控制整个平面的平整程度，公差带为两个平行平面。

- **标注**：`▱ 0.1`
- **检测**：打表法、水平仪、CMM。

### 8.3 圆度与圆柱度
- **圆度**：任一正截面轮廓必须位于半径差为公差值的两同心圆之间。`○ 0.02`
- **圆柱度**：综合控制圆度、直线度和锥度。`⌭ 0.05`

### 8.4 垂直度
控制表面或轴线相对于基准的垂直程度。

- **标注**：`⟂ Ø0.1 Ⓜ A` （孔的轴线相对于基准面A垂直，MMC状态）
- **公差带**：一个直径0.1mm的圆柱体，垂直于基准A。

### 8.5 位置度
最强大的公差，用理论正确尺寸定位公差带中心。

- **标注**：`⌖ Ø0.2 Ⓜ A B C` （孔相对于基准A、B、C在MMC时位置度公差为Ø0.2）
- **解释**：孔的理想位置由带方框的尺寸定义，实际轴线必须落在直径0.2（当孔处于MMC时）的圆柱形公差带内，并且垂直于基准A。
- **MMC补偿**：若孔实际尺寸偏离MMC（即孔做大），位置度公差可等量增大，只要不违反最小实体边界和功能要求。

### 8.6 轮廓度
- **线轮廓度**：`⌒ 0.5 A B` 控制曲线轮廓。
- **面轮廓度**：`⌓ 0.3 A` 控制复杂曲面的形状和位置。

### 8.7 跳动
- **圆跳动**：在旋转一周时，表面固定位置指示器测得的跳动量。`↗ 0.05 A`
- **全跳动**：零件绕基准旋转并轴向移动，在整个表面上跳动量。`↗↗ 0.05 A`

跳动公差综合控制圆度、同轴度、垂直度等多种误差，检测方便，常用于旋转零件。

## 9. 如何阅读 GD&T 标注？一个简单例子

假设图纸上有一个法兰孔标注：
```
位置： ⌖ Ø0.2 Ⓜ A B Ⓜ C
孔径： Ø10 +0.1/0
基准A： 底面平面（垂直面）
基准B： 中心大圆柱外圆（居中）
基准C： 法兰上的一个槽（定向）
```

解读步骤：
1. 先确定基准框架：底面A放置（3点），中心B轴插入（2点），槽口C定向（1点），完全定位。
2. 孔的理论正确位置由带方框的角度和节圆直径定义。
3. 当孔直径正好是其MMC（Ø10，最小孔）时，位置度公差带为直径0.2mm的圆柱体，垂直A，并与B同轴。
4. 如果孔加工到 Ø10.1（LMC，最大实体减材料），则位置度公差可获得0.1的额外补偿，变为Ø0.3。这叫 **奖励公差**。
5. 任何实际孔轴线必须落在该可变的圆柱形公差带内。

## 10. GD&T 入门学习建议

1. **区分“控制什么”**：形状公差<方向公差<位置公差<跳动公差，精度等级与制造成本递增。
2. **理解基准**：基准反映零件的安装和功能方式。
3. **掌握修饰符**：Ⓜ、Ⓛ、Ⓢ 是设计意图的体现，尤其要透彻掌握 MMC 的补偿机制。
4. **结合实际检测**：学习功能检具设计思路，理解“虚拟条件”和“实际边界”。
5. **对比标准差异**：工作中明确是 ASME 还是 ISO，注意规则#1、同心度废除等差异。
6. **练习读图和标注**：从简单轴类、箱体类零件开始，自己画特征控制框。

## 11. 常见误区

- 认为加上 GD&T 就一定更严格。实际上 GD&T 常常放宽了可用公差。
- 混淆基准顺序，导致检测结果不同。
- 忽略公差带形状描述（直径符号Ø的遗漏会导致公差带变为矩形，损失57%的公差区域）。
- 乱用修饰符，例如在不需要补偿的地方标Ⓜ，导致功能失效。
- 将位置度用于控制特征本身的形状（位置度不能代替直线度）。

---

## 总结

GD&T 是精确表达设计意图、保证互换性和功能性的强有力工具。它超越传统正负公差，提供更优的公差带、清晰的基准体系和基于材料条件的补偿。掌握 GD&T 的符号、特征控制框、基准和修饰符，是机械工程师从“画图”走向“设计”的关键一步。

希望这份入门材料能帮你打开 GD&T 的大门，建议配合 ASME Y14.5 标准或相关培训教材深入学习。
