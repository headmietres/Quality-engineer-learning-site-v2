# IQC 实战：来料质量控制的要点与方法

## 1. 引言

在制造业的质量链中，**IQC（Incoming Quality Control，来料质量控制）** 是生产前端的第一道防线。倘若不良物料流入生产线，不仅会造成工时浪费和产能损失，更可能导致批量性成品缺陷，甚至流入客户端。因此，有效的 IQC 不只是入厂检验，更是一个包含策划、抽样、判定、隔离、反馈和供应商改进的完整管理系统。

本指南从实战角度出发，系统拆解 IQC 的职责、流程、方法与关键控制点，帮助制造现场构筑坚固的来料质量屏障。

---

## 2. 什么是 IQC？

**IQC** 是对采购的原材料、零部件、外协件在进入仓库或生产线前进行验证的活动。其核心目的是确保“不合格的物料不接收、不入库、不投产”。

在检验体系中定位：
```
供应商交付 → [ IQC ] → 合格入库 → 生产线 (IPQC) → ...
↓ 不合格
退货/特采/分选
```

与 IPQC、FQC/OQC 的比较：

| 类别 | 对象 | 时机 | 责任 |
|------|------|------|------|
| **IQC** | 外购原材料、零部件 | 入库前 | 拦截上游缺陷，赋能供应商改进 |
| IPQC | 制造过程中的半成品 | 工序间 | 控制内部过程变异 |
| FQC/OQC | 成品 | 入库/出货前 | 防止不良品流出给客户 |

IQC 不只是“检验”，更是连接供应商质量与企业内部需求的关键桥梁。

---

## 3. IQC 的核心目的

1. **拦截缺陷**：杜绝不合格物料上线，避免产线停摆或产生大批量废品。
2. **数据驱动改进**：通过来料不良数据，推动供应商根因分析，降低长期风险。
3. **验证供方能力**：监控供应商的实际交付质量与承诺（如 PPAP 文件、样品）是否一致。
4. **降低总成本**：越早发现缺陷，损失越小。相较线上换件、客户索赔，IQC 阶段失败成本最低。
5. **满足合规要求**：确保材质证明、环保报告、法规认证（如 RoHS、REACH）齐全有效。

---

## 4. IQC 的标准作业流程

一个完整的 IQC 流程通常包括以下步骤：

1.  **来料接收与核对**
    - 仓管员核对送货单与采购单上的物料编码、名称、规格、数量、批次号。
    - 确认供应商是否在合格供方名录内，有无暂停或受限状态。
    - 将物料放置于 **待检区（黄色标识）**。

2.  **文件审查**
    - IQC 检验员查阅随货文件：材质证明、合格证、检测报告、PPAP 文件（首批时）、RoHS 声明等。
    - 若文件缺失或信息不符，直接判为 **文件不合格**，暂停检验，通知采购处理。

3.  **抽样**
    - 根据检验规范中规定的抽样标准（如 ISO 2859 / GB/T 2828.1）、AQL 值，确定样本量并随机抽取样品。
    - 涉及破坏性试验或价值极高的物料，可依据历史质量情况采用特殊抽样方案。

4.  **检验与测试**
    - 按照《来料检验规范》(SIP) 逐项对外观、尺寸、结构、功能、材质等进行检测。
    - 需记录所有测量数据和缺陷描述。

5.  **判定**
    - **接收 (Accept)**：不良数 ≤ Ac，整批允收，贴绿色合格标签，办理入库。
    - **拒收 (Reject)**：不良数 ≥ Re，整批拒收，移入不合格区（红色标识），发起《不合格品报告单》。
    - **特采 (Concession)**：特殊情况经 MRB 评审，可能让步接收，但需按规定审批（必要时须顾客书面同意）。

6.  **标识与入库**
    - 检验完成的物料必须进行状态标识（合格/不合格/让步接收），方可流转。
    - 将检验数据录入系统，并更新供应商质量档案。

7.  **异常处置与闭环**
    - 不合格品执行退货、分选、返工或报废。
    - 向供应商发出 **SCAR (供应商纠正措施要求)**，跟踪改进和效果验证。

## 5. 抽样检验实战要点

IQC 绝大多数场景采用 **计数型抽样检验**，掌握以下要素是确保检验有效且经济的关键。

### 5.1 抽样标准的选用
通常采用 **ISO 2859-1 / GB/T 2828.1**。关键参数定义：
- **批量大小 N**：来料批次的数量。
- **检验水平**：一般选用 **水平 II**，对检验成本低、风险高的产品可选水平 III，反之可选水平 I 或特殊水平 (S-1~S-4)。
- **AQL 值**：基于缺陷分类和客户要求设定，常见参考值：

| 缺陷等级 | 描述 | 常用 AQL 范围 |
|----------|------|---------------|
| 致命缺陷 (Critical) | 影响安全、法规 | 0，或 0.01~0.1，通常不允许出现 |
| 严重缺陷 (Major) | 影响功能或装配 | 0.25, 0.40, 0.65, 1.0 |
| 轻微缺陷 (Minor) | 外观等不影响使用 | 1.0, 1.5, 2.5, 4.0 |

当产品同时包含多个缺陷等级时，应分别设定 AQL 并分别判定，任一等级不达标则整批拒收。

### 5.2 样本量与接收/拒收数
使用标准化抽样表确定样本量字码，再根据 AQL 找到对应的 **Ac（接收数）** 和 **Re（拒收数）**。

示例：来料 1000 件，检验水平 II，AQL 严重=0.65，轻微=1.0。
- 字码 J，样本量 80。
- 严重缺陷 AQL=0.65 → Ac=1, Re=2。
- 轻微缺陷 AQL=1.0 → Ac=2, Re=3。
若检出 1 个严重缺陷和 2 个轻微缺陷：严重缺陷数=1 (≤1) 通过，轻微缺陷数=2 (≤2) 通过 → 整批接收。若严重缺陷=2 (≥2)，则直接拒收，不用看轻微缺陷。

### 5.3 抽样转移规则应用
应当根据供应商的持续质量表现，动态调整抽样严格度：
- **正常→加严**：连续 5 批中 2 批在正常检验被拒收。
- **正常→放宽**：过程质量一贯优于 AQL，且生产稳定。样本量通常减半。
- **加严→正常**：连续 5 批加严接收后恢复。
- **暂停检验**：加严检验时未达到要求，且累计不合格批次达到 5 批，应暂停该供方此产品检验，直到其改善为止。

这一机制迫使供应商因惧怕加严或被停供而主动维持质量。

## 6. 来料检验的具体内容与方法

检验依据的核心文件是 **SIP（Standard Inspection Procedure，标准检验规程）**，其中详细定义了检验项目和方法。

### 6.1 外观检验
- **工具**：目视、放大镜、光源箱、标准样件、色板。
- **项目**：颜色、光泽、缩水、毛刺、划痕、油污、锈蚀、标签内容与位置。
- **实战技巧**：使用 **限度样品 (Boundary Sample)** 作为允收/拒收的实物标准，减少主观判断差异。

### 6.2 尺寸检验
- **工具**：卡尺、千分尺、通止规、螺纹规、高度规、二次元/三次元影像测量仪。
- **要点**：
  - 识别图纸上的关键尺寸 (KPC) 和非关键尺寸，优先保证关键尺寸的准确性。
  - 对于定位基准、配合尺寸，必须使用与装配功能匹配的检具或方法。
  - 巡检抽检时，测量点应覆盖不同穴号（针对多腔模具）。

### 6.3 材质与性能测试
- **材质验证**：核对材料证明（Mill Certificate），必要时用光谱仪、碳硫分析仪抽查或送第三方检测。
- **硬度/强度**：硬度计、拉力试验机。
- **电性能/功能**：耐压测试、导通测试、功能模拟。
- **环境合规**：定期抽检或核验供应商提供的 RoHS、REACH 报告。

### 6.4 结构组装与匹配性
- 将部件进行试装配，检查配合间隙、干涉、顺畅度。特别是首件或新模件，必须进行此项验证，单检尺寸可能无法发现装配问题。

## 7. 来料检验的基础设施

### 7.1 检验区域规划
物理区域必须严格三区分开并带明显标识：
- **待检区**：黄色框线/地贴，临时存放待验品。
- **合格品区**：绿色标识，已放行物料。
- **不合格品区**：红色标识并有物理隔离，防止误用。

### 7.2 量具与测量系统
- 所有用于接收判定的量具必须在校准有效期内，并带有校准标识。
- 关键特性的测量系统必须通过 **GR&R**（重复性和再现性）评估，确保测量结果可靠。
- 建立量具日常点检记录，尤其使用频繁的通止规、卡尺。

### 7.3 信息化系统
使用 ERP/QMS 系统记录 IQC 检验数据，实现：
- 批次追溯：追溯到来料批号、供应商生产日期、检验员。
- 自动判定：根据抽样表和缺陷数系统自动决定接收/拒收，减少人为错误。
- 供应商质量报告：自动统计来料批次合格率、PPM、主要缺陷分布。

## 8. 不合格品的处理与特采

### 8.1 常规处理流程
IQC 拒收后，立即：
1.  不合格批次挂红牌，移入 MRB 区。
2.  发出《来料不合格处理单》，启动 MRB 评审。
3.  评审结果常见处置：**退货**、**分选**（由供应商或第三方挑选，合格品入库）、**返工/返修**（供应商负责）、**报废**（本地报废，供应商承担损失）。
4.  处置后必须由 IQC 重新检验合格，才能接收。

### 8.2 特采 (Waiver/Concession)
生产急需且不合格项不影响最终产品功能、安全时，可由申请部门提出特采申请：
- 必须经过跨部门评审（设计、工艺、质量），确认风险可控。
- 若该物料涉及客户的特殊特性，必须获得 **客户书面让步接收** 方可实施。
- 特采物料必须在整批及流转单据上做醒目标识，以便追溯。
- 特采必须限定批次、数量，并设定失效日期，禁止常态化滥用。

## 9. 从检验到供应商改进的闭环

IQC 的核心价值不仅在于“拦截”，更在于“推动源头改善”。

- **数据反馈**：每月向供应商通报批次合格率、PPM 和 Top 缺陷问题，并与目标对比。
- **8D 报告要求**：对重复性问题或严重缺陷，要求供应商在期限内提交根本原因分析和纠正措施报告。
- **现场审核**：对持续表现不佳的供应商进行过程审核（VDA 6.3 或类似），验证改进效果。
- **质量协议**：将 IQC 抽样方案、AQL 要求、不合格处置流程约定在供货合同或质量协议中，权责清晰。
- **免检管理**：对于长期表现卓越、过程能力稳定、已建立稳健防错体系的供应商，可升级为 **免检** 或 **放宽检验**，但仍然需要定期验证（如船运分析、周期性抽检）防止质量退化。

## 10. 常见实战误区与应对

| 误区 | 风险 | 应对策略 |
|------|------|----------|
| **所有物料同一个AQL** | 关键件漏检风险高，一般件过度检验浪费资源。 | 依据物料分类（关键/重要/一般）分级设定AQL。 |
| **只检尺寸不检匹配** | 尺寸合格但装不上，生产线大批停工。 | 对涉及装配的物料，定期做真实组装的适配检验。 |
| **文件审查流于形式** | 材质证明造假或过期，出现安全/法规风险。 | 定期核验证书真伪，对关键材料进行第三方抽检。 |
| **库存积压时跳过IQC直接入库** | 上线后发现不良，返工难度倍增。 | 设定红牌物料制度，无 IQC 合格标签禁止发料。 |
| **IQC数据不录入系统** | 无法评估供应商质量，问题重复发生。 | 当天完成数据录入，设定逾期未检或未录的警报。 |
| **让步接收常态化** | 供应商有恃无恐，来料质量整体下滑。 | 限定特采次数（如一年内≤2次），超出后降级供应商。 |
| **检验员依赖记忆判断** | 标准不统一，漏判误判多。 | 利用SIP图文并茂，限度样品实物化，定期做检验员一致性比对。 |

## 11. 总结

**IQC 是供应链质量管理的前沿阵地。** 做好来料质量控制的关键在于：

- **策划优先**：基于物料风险和功能等级制定科学的抽样方案和检验规范。
- **标准明确**：SIP、限度样品、缺陷定义必须唯一且可操作，确保检验员判断一致。
- **数据驱动**：将每一次检验结果都转化为供应商改进的动力，实现数据闭环。
- **严守底线**：不合格品隔离、特采审批流程决不可因交付压力而放松，质量是生产的保障而非障碍。
- **持续优化**：根据供应商表现动态调整检验方案，推动优秀供方迈向免检，将资源聚焦于高风险物料。

只有将 IQC 从单纯的“挑拣坏品”升华为“管理来料质量风险与源头改进”的体系，才能真正为后端的精益制造和零缺陷目标奠定坚实基础。
